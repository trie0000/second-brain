STEP 1

Excelブックを接続する

  1. 「読み込み」 をクリック（「データの変換」は次のステップで）
STEP 2

年月列を日付型に変換（必須）

リボン「ホーム」→「データの変換」を操作します
Power Query エディターが開きます
  1. リボン 「ホーム」 → データの変換
  2. 左ペインで 「メトリクステーブル」 を選択
  3. 「年月」列のヘッダー左の型アイコン（ABC）をクリック → 「日付」 を選択
  4. 「現在のものを置換」を選択
  5. 左ペインで 「コメントテーブル」 を選択して同じ操作
  6. リボン 「閉じて適用」
この変換をしないとグラフのX軸が「1月→10月→11月…」という文字順になります。必ず実施してください。

02 — DAXメジャー 4 min
右ペイン「データ」タブ → メトリクステーブルを右クリックします
「メトリクステーブル」 を右クリック → 「新しいメジャー」 → 数式バーにコードを貼り付け → Enter
メジャー ①（必須）

直近12ヶ月フラグ — 全グラフに適用するフィルター

直近12ヶ月フラグ = VAR MaxDate = CALCULATE(MAX('メトリクステーブル'[年月]), ALL('メトリクステーブル')) VAR MinDate = DATE(YEAR(MaxDate), MONTH(MaxDate) - 11, 1) RETURN IF(MAX('メトリクステーブル'[年月]) >= MinDate, 1, 0)
各グラフのビジュアルフィルターに「直近12ヶ月フラグ が 1 である」を追加すると、月スライサーで選んだ月を基準に直近12ヶ月が自動表示されます。
メジャー ②（任意）

前月比 — グラフタイトル横の増減表示用

前月比_新規通知 = VAR 今月 = CALCULATE(SUM('メトリクステーブル'[新規通知数]), DATESMTD('メトリクステーブル'[年月])) VAR 先月 = CALCULATE(SUM('メトリクステーブル'[新規通知数]), DATEADD(DATESMTD('メトリクステーブル'[年月]), -1, MONTH)) RETURN 今月 - 先月

07 — スライサー設定

スライサーは各ページ共通です。1ページ目で作成後、コピーして他のページに貼り付けます。

月スライサー

対象月ドロップダウン（単一選択）

視覚化パネル → 🔘 スライサーを配置します
スライサーアイコンをクリック → フィールドに メトリクステーブル[年月] をドラッグ
設定箇所操作
スライサーの種類右ペイン「書式」→ スライサーの設定 → 「ドロップダウン」
選択の種類「選択の種類」→ 「単一選択」 をオン
単一選択にすることで「選択した月のコメント」が正確に1件だけ表示されます。ドロップダウンにすると画面スペースも節約できます。
地域スライサー

地域チェックボックス（複数選択）

設定値
フィールドメトリクステーブル[地域]
スライサーの種類リスト（デフォルトのまま）
選択の種類複数選択（デフォルトのまま）
他のページへのコピー

スライサーを2・3ページ目にも貼り付ける

  1. 月スライサーと地域スライサーを両方選択（Ctrl+クリック）
  2. Ctrl+C
  3. 🟠 ハイリスクポート検知ページのタブをクリック
  4. Ctrl+V（同じ位置に貼り付けられます）
  5. 🔵 管理画面公開検知ページでも同じ操作
「貼り付け」すると他ページのビジュアルとも自動連動します。再設定は不要です。

Power BI Desktop での更新手順

リボン「ホーム」→ 更新ボタンをクリックします
🔄 更新 をクリック → Excelの最新データが読み込まれます
  1. Excel に当月データを追記して保存
  2. Power BI Desktop で .pbix を開く
  3. リボン 「ホーム」 → 🔄 更新
  4. Ctrl+S で .pbix を上書き保存
  5. 最新の .pbix を Teams チャネルに上書きアップ
閲覧者は Teams から最新の .pbix をダウンロードして開くだけです。Excelへのアクセス権は不要です。
